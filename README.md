# Unity Platformer Study Project

A simple Unity platformer created as an academic study project.  
The project was led by Olivia for university coursework, and I contributed most of the game assets and supported the project through GitHub.

This repository is included as a small portfolio item because it shows practical exposure to Unity project structure, asset integration, version control, and collaborative game development workflow.

## My Contribution

- Contributed most of the visual/game assets used in the project.
- Supported the project through GitHub repository workflow.
- Worked with Unity project structure, including Assets, Scenes, Packages, and Project Settings.
- Gained practical exposure to how game features, assets, scenes, and testing considerations fit together in a small platformer project.

## How to Open

In github desktop top left corner File -> Clone a Repository -> URL - insert github url of  the repository folder.

In unityhub Projects -> top right corner -> Add -> Add project from disc -> select folder with project - the whole folder that was downloaded from github

install appropriate version of unity, remove other version of unity.
